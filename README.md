# Lifecycle-Messaging-System
This is a code bundle for Lifecycle Messaging System. 
A trigger-based lifecycle system that moves Activation, TTFV, Retention, conversion and reduces churn.

## Running the code

1) Install Node.js
2) Run `npm i` to install the dependencies.
3) Run `npm run dev` to start the development server.
